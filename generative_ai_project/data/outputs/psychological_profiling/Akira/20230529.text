1.1 TACTICAL ANALYSIS
- Identify 3-4 key tactics employed by the victim.
- Classify each tactic using the provided list (A-E).
- Provide textual evidence for each classification.

1.2 STRATEGY EVALUATION
- Evaluate the effectiveness of each tactic.
- Rank tactics from most to least effective (1-3).
- Justify rankings with textual evidence.

1.3 TACTIC VS. STRATEGY
- Distinguish between TACTIC (specific action) and STRATEGY (overall approach).
- Example: "Poverty pleading" (Tactic) vs. "Poverty pleading strategy" (Strategy).
