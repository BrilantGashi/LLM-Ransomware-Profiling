1.1 TACTICAL MARKERS
- Identify linguistic markers of negotiation tactics (e.g., "Poverty pleading," "Technical stalling," "Relationship-building," "Defiance," "Compliance").
- Mark each message with a tactical label (A-E) and provide justification.

1.2 TACTICAL EFFICIENCY
- Rate each tactic's effectiveness (1-5 scale): 1=Failed; 5=Highly effective.
- Justify ratings with textual evidence.

1.3 TACTICAL DYNAMICS
- Identify shifts in tactics over time.
- Correlate tactic shifts with negotiation outcomes (e.g., price concessions, negotiation collapse).

1.4 TACTICAL DEPLOYMENT
- Map out the sequence of tactics used by both parties.
- Identify any patterns or cycles in tactic deployment.

1.5 TACTICAL SYNERGY
- Identify instances where tactics from both parties synergize (e.g., attacker's empathy appeal + victim's compliance).
- Analyze the impact of these synergies on negotiation outcomes.

1.6 TACTICAL INCONSISTENCIES
- Identify any inconsistencies or contradictions in tactics.
- Analyze the impact of these inconsistencies on negotiation outcomes.
